# 청약 AI 상담봇

## Vercel 배포 방법 (5분)

### 1. GitHub에 올리기
```bash
git init
git add .
git commit -m "청약 AI 상담봇"
git remote add origin https://github.com/YOUR_USERNAME/subscription-chatbot.git
git push -u origin main
```

### 2. Vercel 배포
1. https://vercel.com 접속 → GitHub 로그인
2. "Add New Project" → GitHub 저장소 선택
3. 설정 그대로 → "Deploy" 클릭
4. 배포 완료! 링크 생성됨 (예: https://subscription-chatbot.vercel.app)

## 로컬 실행
```bash
npm install
npm run dev
```
